# 📋 项目总结

## 🎯 项目目标
解决机场订阅链接5分钟失效的问题，通过GitHub Actions自动抓取和更新订阅，提供稳定的raw链接。

## 🏗️ 项目架构

### 核心组件
```
机场订阅自动抓取器
├── 单订阅模式 (subscription_fetcher.py)
├── 多订阅模式 (multi_subscription_fetcher.py)
├── 配置测试 (test_config.py)
├── 一键部署 (install_and_build.bat/.sh)
└── GitHub Actions 自动化
```

### 工作流程
```
机场订阅 (5分钟失效)
    ↓
GitHub Actions (每4分钟抓取)
    ↓
更新GitHub仓库文件
    ↓
生成稳定的raw链接
    ↓
供代理工具使用
```
